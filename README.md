Step 1: Clone the project by copying the link.<br>
Step 2: Checkout to the master branch.<br>
Step 3: Composer update.<br>
Step 4: Add an env file. Set database name = student_management.<br>
Step 5: Create a database named student_management.<br>
Step 6: Create an application key - php artisan key:generate.<br>
Step 7: Run the migrations - php artisan migrate.<br>
Step 8: Run the seeders - php artisan db:seed.<br>
Step 9: Run the project - php artisan serve.<br>
